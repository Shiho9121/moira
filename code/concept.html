Concept

Moira

「運命の糸を紡ぐ女神」

ギリシャ神話における「運命の三女神」をイメージしております。
この香りのレースを纏えば、あなたの運命の糸を紡ぎます。
そして、あなたの気品と大人の魅力を表現し、目にするものを魅了します。
Moiraは、強くて甘い現代女性に勇気を与える香水です。
プライベートシーンやリラックスシーンに、ぜひご使用ください。

Point

手首のほかにも

手首以外にも、足首や腹部の下あたりに1プッシュがおすすめです。 香りが下から上へのぼり、優しい香りの演出ができます。

レースのように

優しくレースのような繊細な香りで、あなたを包み込みます。

コンパクト

Moiraの製品はコンパクトな作りになっております。常にカバンや ポーチに入れておくことができます。
